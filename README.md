# Hangman Game

## Project Description
This project is a simple text-based Hangman Game developed using Python. The player guesses a randomly selected word one letter at a time. The player has a maximum of 6 incorrect guesses to find the hidden word.

## Features
- Randomly selects a word from a predefined list of five words.
- Allows up to 6 incorrect guesses.
- Displays the current progress of the hidden word.
- Keeps track of guessed letters.
- Validates user input.
- Beginner-friendly console application.

## Technologies Used
- Python 3
- Random Module
- While Loop
- If-Else Statements
- Lists
- Strings

## How to Run

1. Clone or download this repository.
2. Open the project in Visual Studio Code.
3. Open the terminal.
4. Run the following command:

```bash
python hangman.py
```
